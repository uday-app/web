# Stack Defaults

- Next.js 16 App Router
- React 19
- Tailwind CSS v4
- HeroUI v3

## Defaults

- Server Components first.
- Server Actions for writes.
- Fresh runtime data by default.
- Cache only with explicit `use cache`, `cacheLife`, or `cacheTag`.

## Avoid

- TanStack Query
- TanStack Table
- TanStack Form
- Extra client data orchestration unless explicitly requested
